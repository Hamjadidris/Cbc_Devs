# Cbc_Devs
Open source Project for my team at the first CBC full stack training!

Hello so we the devs at CBC decided to come up with this project, (CBC website) We basically gonna put to 
practice all we have learnt so far and see how just fine and really fast we have grown from level 0 no stack level. 
Being a devleoper is fun and really worthwhile so we intend to put all that energy on this, feel free to contibute 
follow steps bellow!!.

To Contribute here(Version control) Step 1: Click on Fork at the top right corner Step 2: Clone your forked repository Step 3: cd into the cloned folder | Cbc_Devs Step 4: git remote add upstream https://github.com/chisombiri/Cbc_Devs Step 5: git pull upstream "branch" Step 6: Check out to the task branch | git checkout -b <NAME_OF_THE_TASK> Creating a pull request(when done with your code/changes) Step 1: Run: git add . Step 2: Run: git commit -m "" Step 3: git push origin <BRANCH_NAME> Go to the repository https://github.com/chisombiri/Cbc_Devs As soon as you get there, you are going to see a green ‘compare and create a pull request’ Click on it, and type your message, click on create pull request Thanks to: @chisombiri
